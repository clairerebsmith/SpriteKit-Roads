Changing alien colors

If they player chooses to match only one alien, we’re going to penalize them by completely changing all the alien colors, effectively resetting the game grid.

Making this change is another chance for you to use SKTexture. We used it previously to change one type of alien into another type, but here we’re going to use it to change everything.

Here’s what we’re going to do:

  1. Create an array of all the aliens image names we can use.
  2. Loop over all column, and all items in each column.
  3. Use a random image name for our new item name.
  4. Create an SKTexture from that image name, and use that for our item’s texture.

Replace your current penalizePlayer() method with this:

func penalizePlayer() {
    let itemImages = ["alien-blue", "alien-gray", "alien-green", "alien-pink", "alien-purple", "alien-yellow"]

    for col in cols {
        for item in col {
            let changeTo = itemImages.randomElement()!
            item.name = changeTo
            item.texture = SKTexture(imageNamed: changeTo)
        }
    }
}

And that’s it – try selecting a single alien now and you should see the rest of the grid change!
