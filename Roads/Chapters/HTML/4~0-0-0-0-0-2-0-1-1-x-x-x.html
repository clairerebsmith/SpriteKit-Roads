Changing shape colors

If they player chooses to match only one shape, we’re going to penalize them by completely changing all the shape colors, effectively resetting the game grid.

Changing our shapes is actually quite easy, because each sprite node has a texture property that contains its image. SpriteKit textures are handled using a class called SKTexture, and you can create them just like creating an SKSpriteNode then assign them to an existing sprite node without having to delete and re-add it.

Here’s what we’re going to do:

  1. Create an array of all the shapes image names we can use.
  2. Loop over all column, and all items in each column.
  3. Use a random image name for our new item name.
  4. Create an SKTexture from that image name, and use that for our item’s texture.

Replace your current penalizePlayer() method with this:

func penalizePlayer() {
    let itemImages = ["shape-circle", "shape-diamond", "shape-heart", "shape-pentagon", "shape-square", "shape-star", "shape-triangle"]

    for col in cols {
        for item in col {
            let changeTo = itemImages.randomElement()!
            item.name = changeTo
            item.texture = SKTexture(imageNamed: changeTo)
        }
    }
}

And that’s it – try selecting a single shape now and you should see the rest of the grid change!
