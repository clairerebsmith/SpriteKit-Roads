Adjusting their move count

If they player chooses to match only one balloon, weâ€™re going to penalize them by subtracting from their remaining move count. How much to subtract is down to you, but it needs to be severe enough to dissuade them from doing it unless it allows an incredible combo.

In my testing I found 9 to be the magic number, so all we need to do is subtract 9 from movesRemaining inside the penalizePlayer() method. Put this in there now:

movesRemaining -= 9

In case you were wondering, 9 is a good choice because we already subtract 1 for every move the player makes. This means if they match only one balloon they will actually lose 10 moves, which works well.
