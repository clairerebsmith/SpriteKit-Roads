Changing balloon colors

If they player chooses to match only one balloon, we’re going to penalize them by completely changing all the balloon colors, effectively resetting the game grid.

Changing our balloons is actually quite easy, because each sprite node has a texture property that contains its image. SpriteKit textures are handled using a class called SKTexture, and you can create them just like creating an SKSpriteNode then assign them to an existing sprite node without having to delete and re-add it.

Here’s what we’re going to do:

  1. Create an array of all the balloons image names we can use.
  2. Loop over all column, and all items in each column.
  3. Use a random image name for our new item name.
  4. Create an SKTexture from that image name, and use that for our item’s texture.

Replace your current penalizePlayer() method with this:

func penalizePlayer() {
    let itemImages = ["balloon-black", "balloon-blue", "balloon-green", "balloon-purple", "balloon-red", "balloon-yellow", "balloon-orange"]

    for col in cols {
        for item in col {
            let changeTo = itemImages.randomElement()!
            item.name = changeTo
            item.texture = SKTexture(imageNamed: changeTo)
        }
    }
}

And that’s it – try selecting a single balloon now and you should see the rest of the grid change!
