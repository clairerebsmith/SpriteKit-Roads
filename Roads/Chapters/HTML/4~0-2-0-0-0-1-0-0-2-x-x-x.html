Adjusting the clock

If they player chooses to match only one alien, we’re going to penalize them by subtracting from their remaining time. How much to subtract is down to you, but it needs to be severe enough to dissuade them from doing it unless it allows an incredible combo.

In my testing I found 10 seconds to be the magic number, so all we need to do is subtract 10 from gameStartTime inside the penalizePlayer() method. Put this in there now:

gameStartTime? -= 10

The syntax is a little strange because gameStartTime is optional – what it means is “if this has a value subtract 10 from it, otherwise do nothing.”

In case you were wondering, we subtract 10 because we’re effectively saying “make it look like the game started 10 seconds earlier than it did,” which causes 10 seconds more to have passed to reach the current point.
