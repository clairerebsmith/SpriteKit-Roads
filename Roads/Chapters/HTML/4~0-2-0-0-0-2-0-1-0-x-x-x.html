Subtracting from their score

If they player chooses to match only one alien, we’re going to penalize them by subtracting from their score. How much to subtract is down to you, but it needs to be severe enough to dissuade them from doing it unless it allows an incredible combo.

In my testing I found 1000 to be the magic number - just enough to hurt players, but still worth doing if it lets them match 10 or more aliens as a result.

So, put this inside your penalizePlayer() method:

score -= 1000

Done!
