Wrap up

That’s the end of the fourth project, and I hope you feel you’ve learned a lot! In this project you:

  • Drew sprites on the screen with SKSpriteNode.
  • Added animation with SKAction.
  • Played looping audio with SKAudioNode.
  • Drew text with SKLabelNode.
  • Read screen touches and looked up what nodes were tapped.
  • Used GCD to launch a new game after some time has passed.

And more – you should be proud of your progress!

Match three games are hugely popular around the world, and hopefully you already have a few ideas how you could make this better. If not, here are some homework ideas to get you started…

Homework

For an easy challenge, try adding extra moves for the player if they manage to match 16 items all at once.

For something more challenging, stop the music when the game ends. This is best done by making our background music into a property, then calling run() on it with the action SKAction.stop() inside endGame().

And for a real mental work out, see if you can write a method that recommends a move to players if press a Hint button. A simple way of implementing this would be to check the match count for all items in the grid, then show a yellow square over the item with the most matches.
